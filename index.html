Water Leak Stop



The WLS-1, is a system for detection and management of water leaks. In such a case it will close the water supply and notify you immediately.

   

WLS-1 Main Features

  • Suitable for installation in switch cabinet (35mm Din-Rail)
  • Standalone or in conjunction with an alarm system
  • Alarm by: monitoring the continuous flow time or leakage
  • Automatic management of water supply (SPDT Relay)
  • Internal buzzer warning / alarm output (SPDT Relay)
  • Two types of flow sensors (Hall Effect flow sensor / flow switch N.O.)
  • Ultra-sensitive water detection circuit
  • Remote control (via external GSM or button)
  • Provision for occasional large consumptions (pools etc.)
  • Compatible with many types of water sensors (contact probe, cable)

The WaterLeakStop WLS-1, is an integrated system which is able to detect and manage a water leak event. Provides monitoring and management of mains water in a hydraulic network or a portion thereof, by monitoring the time period of continuous flow and comparing this with the user settings. Moreover, it allows the detection of water leaks in selected locations, with the use of local water sensors.

By using a suitable flow sensor, at the starting point of a pipe network and an electric valve (or solenoid valve), it is capable to handle the water flow and in accordance with the initial configuration of the user (Maximum Continuous Flow Time - MCFT), to remove or reduce the risk, of extensive damage from a leak. The measurement accuracy depends only by the flow sensor / switch sensitivity to be used, giving the flexibility to the user to choose one, in accordance with the requirements of the application. It support two sensor types. The first type is a flow switch (normal open contact). The second type is a flow sensor, with pulse output 12V (Hall Effect). Sensors, where use similar output commands during detection, can be used.

The alarm where produced by the flow detection mode, is based in completion of the MCFT. If this time period overcome, the system will go to alarm mode. The MCFT selected by the user, between eight (8) preset values (5, 10, 20, 30, 45, 60, 90, 150 min). The direct management of the water supply and the alarm notification will operate also if water leak detected, from the corresponding zone of water sensors.

An alarm will also triggered in case of water leak detection, from the corresponding zone of water sensors. This zone responds directly (with 4sec verification time). More than one sensors can be connected in parallel. Compatible sensors are any type with Normally Open contact.

The user can easily handle the main electric valve from the device, by pressing the corresponding key. Remote management is provided by connecting an external button or connecting to a PGM, from an already installed system alarm or a GSM device.

Benefits:

  • Dual supervision:
    • Βy monitoring the maximum allowed time, of continuous flow.
    • By monitoring sensitive areas through humidity sensors of various types (Pin, cable, etc.).
  • Single supervision:
    • Flexible, because it can operate based only on water sensor zone or only in maximum flow time.
  • Control the main water supply via electric valve.
  • One cycle bypass operation, for seamless use of long periods of continuous flow, such as filling a swimming pool or a garden watering system.
  • Integration with an existing alarm system, giving the following advantages:
    • Perform operations such as the transmission of the alarm.
    • Remote control of the electromagnetic valve

Additional features supported:

  • Connection for an indoor / outdoor siren or for a trigger signal to an alarm system.
  • Connection for a PSTN phone dialer (only notification) or a GSM phone dialer (notification and remote control function).
  • Connection for an external transmitter that will transmit the information of the output signals. For example using a transmitter, compatible with the alarm system of the house, you can transmit a leak event to it, without cables.
  • The use of WDT (watchdog timer) and special firmware, ensures the proper functioning of the device.

Features:

  • Start flow detection (lt/min): According to the sensor's sensitivity.
  • Flow sensors: Flow switch (NO), Flow sensor (Hall Effect type 12VDC).
  • Water leak sensors: Points detection probe, sensor cable.
  • Flow management: A SPDT relay, manage a solenoid or electrical water valve.
  • Remote control (optional): Reset, Open/Close solenoid water valve, exclude electric valve.
  • Notification: Internal buzzer, SPDT relay to trigger an external siren, GSM module, house alarm etc.
  • MCFT table: 8 different options (5, 10, 20, 30, 45, 60, 90, 150).
  • Modes: Normal, Bypass 1 Cycle, Exclude Valve (only notification).
